USSD

Posted by Dhruv Upadhyay on 01/01/2018

In my 3rd semester, I developed the first simple project called "USSD"- Unlimited Safe Secure Drive. This software implemented in .Net framework using C#. This application provides a feature for uploading and downloading a specific file at a particular time. In this application user must select the file and set time for uploading and downloading. Backend API's work for that file and when time is bound then file automatically uploaded and downloaded with user get a notification. This application based on windows. It is my first simple project, that I developed and stimulate my interest in cloud computing.
